
サンプル２

○目標
  Beanの様々なプロパティに、汎用的にアクセスしてみよう。
  
  Beanには、様々なアクセサがあります。
  Javaのプログラム上で、それらのアクセサを使うのは、問題ないのですが、
  Java言語ではない表現で記述される実行環境で、それらのアクセサに対するアクセスを
  簡単で画一的な表現を使ってアクセスするJavaライブラリを作りたい場合があります。
  
  一般的には、JSPのタグライブラリのように、HTMLという構造化言語の中に、JVM中で
  動作するタグライブラリを記述して、JavaのBeanのプロパティにアクセスして、
  HTMLにその値を出力したい場合などがあります。
  
  Nimbusで言えば、サービス定義でのアクセサを使ったインジェクションであったり、
  BeanFlow定義でのインジェクションやメソッドの実行などです。
  
  Nimbusでは、これらの要求に応えるために、簡単で画一的な記述方法で、Beanの様々な
  プロパティに対して、同じインタフェースでアクセスする機能を提供しています。

○リソース一覧
  ・サンプル実行クラスのソースファイル
    Main.java

○用意するもの
  ・NimbusのJar

○実行
  （１）コンパイルする。
      例：
        javac -classpath nimbus.jar Main.java
   
  （２）Mainクラスを実行する。
      例：
        java -classpath .;nimbus.jar Main
