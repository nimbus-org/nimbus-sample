
サンプル３

○目標
 データセットを使って、FLV（Fixed Length Value：固定長）ファイルを読み書きしてみよう。
 jp.ossc.nimbus.beans.dataset.Recordを経由する事で、データを扱いやすくする。

○リソース一覧
  ・サンプル実行クラスのソースファイル
    Main.java

○用意するもの
  ・NimbusのJar

○実行
  （１）コンパイルする。
      例：
        javac -classpath nimbus.jar Main.java
   
  （２）Mainクラスを実行する。
      例：
        java -classpath .;nimbus.jar Main
